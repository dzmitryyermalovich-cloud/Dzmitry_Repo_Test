# Гимназия № 23 (

- Прежние названия: Средняя школа № 23 (1947-2002), Еврейская школа № 23 (1922-1931), Политехническая фабрико-заводская школа № 23 (1931-1936), Лётное училище (1936-1947)
- Основана: 1922
- Преобразована в гимназию: 2002
- Директор: Ирина Леонидовна Пасюкович
- Тип: Гимназия
- Количество учеников: 433
- Адрес: Республика Беларусь, г. Минск, проспект Независимости, д. 45
- Сайт: gymn23.minsk.edu.by

### Краткое описание:
Гимназия № 23 в Минске – это образовательное учреждение, которое прошло через несколько этапов своего существования, начиная с 1922 года. Она была известна под разными названиями и играет важную роль в образовательной системе города. Сегодня гимназия предоставляет обучение для более чем 400 учеников и управляется директором Ириной Леонидовной Пасюкович.
